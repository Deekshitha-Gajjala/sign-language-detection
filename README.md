# Sign Language Detection System

## Overview

The Sign Language Detection System is a computer vision application that recognizes hand gestures and sign language symbols in real time using machine learning and image processing techniques.

The project aims to bridge communication gaps by enabling computers to understand sign language gestures and convert them into meaningful outputs.

---

## Problem Statement

Millions of people rely on sign language as their primary mode of communication. However, communication barriers often arise when interacting with individuals who do not understand sign language.

This project leverages computer vision and machine learning to recognize hand gestures and improve accessibility through automated sign language interpretation.

## System Workflow

1. Capture live video stream.
2. Detect hand landmarks and gestures.
3. Extract relevant features.
4. Process features through machine learning model.
5. Predict corresponding sign language symbol.
6. Display output to the user.
